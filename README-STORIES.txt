Then commit and let Render redeploy.

To add more stories:
1. Open the stories folder in GitHub.
2. Create a new file like story-04-your-title.txt
3. Put Khmer story text inside.
4. Commit changes.
5. Render redeploy.

Keep stories adult romantic / husband-wife / suggestive, not graphic.
